## 安装方法
1. 在Unity编辑器中，选择 Assets > Import Package > Custom Package
2. 选择 unity-ai-mcp-trae-v1.9.146.unitypackage 文件
3. 在导入对话框中确认要导入的文件
4. 点击 Import 按钮完成安装
